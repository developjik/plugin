# Frontend Bad Example

## Context
Marketing page generated from generic defaults.

## What Bad Looks Like
- White cards, weak branding, and interchangeable layout.
- Purple gradient hero with no product-specific point of view.
- Visual polish is mistaken for originality.

## Expected Evaluation
- design quality: low
- originality: low
- craft: medium
- functionality: medium

## Why This Matters
- Use this as a negative anchor when the evaluator praises generic output.
