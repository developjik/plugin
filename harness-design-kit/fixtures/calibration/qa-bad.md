# QA Bad Example

## Context
Evaluator read code but did not exercise the running system.

## What Bad Looks Like
- Broad praise without reproduction steps.
- “Should work” language instead of tested behavior.
- No screenshots, logs, or user-flow notes.

## Expected Evaluation
- verdict: fail or revise
- findings: note lack of evidence as a defect

## Why This Matters
- Use this when the evaluator confuses code review with QA.
